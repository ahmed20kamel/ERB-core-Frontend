# Create OG Image for Social Media

The OG image should be 1200x630px and include:
- AL YAFOUR CONSTRUCTION logo
- "AL YAFOUR – Procurement System" text
- Professional design matching brand colors

## Method 1: Using Design Tool
1. Open Figma, Canva, or Photoshop
2. Create 1200x630px canvas
3. Add logo (centered or top-left)
4. Add text: "AL YAFOUR – Procurement System"
5. Use brand colors (#F97316 for primary)
6. Export as PNG: `og-image.png`

## Method 2: Using HTML/CSS (Node.js script)
Create a script to generate OG image using canvas or puppeteer.
