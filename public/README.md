# Public Assets Directory

This directory contains all public assets for the Procurement System.

## Required Files

### Logo Files
- `logo.svg` - Main logo (SVG format, recommended for all use cases)
- `logo.png` - Main logo (PNG format, 80x80px recommended)
- `logo-dark.png` - Dark mode logo variant (PNG format)
- `logo-light.png` - Light mode logo variant (PNG format)

### Favicon Files
- `favicon.ico` - Main favicon (16x16, 32x32, 48x48 sizes)
- `favicon.svg` - SVG favicon (modern browsers)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon

### Apple Touch Icons
- `apple-touch-icon.png` - 180x180px for iOS devices

### Android Icons
- `android-chrome-192x192.png` - 192x192px for Android
- `android-chrome-512x512.png` - 512x512px for Android

### Other Files
- `og-image.png` - Open Graph image (1200x630px recommended)
- `safari-pinned-tab.svg` - Safari pinned tab icon
- `site.webmanifest` - Web app manifest
- `robots.txt` - Search engine robots file
